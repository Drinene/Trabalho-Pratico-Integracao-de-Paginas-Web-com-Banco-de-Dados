# Livros Next - Catálogo de Livros

Sistema de catálogo desenvolvido em Next.js integrado com MongoDB.

## Como Usar

1. **Instalar**: `npm install`
2. **Executar**: `npm run dev`
3. **Acessar**: `http://localhost:3000`

## Requisitos

- Node.js 16+
- MongoDB
- Servidor Express na porta 3030

## Funcionalidades

- Listar livros
- Cadastrar livros  
- Excluir livros
- Interface responsiva

## API

Consome endpoints REST em `http://localhost:3030/livros`
